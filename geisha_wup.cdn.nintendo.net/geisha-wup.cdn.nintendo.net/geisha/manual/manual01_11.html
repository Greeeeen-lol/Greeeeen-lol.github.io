Entering Download Codes

Enter a download code to download software.

What is a Download Code?

A 16-character alphanumeric code used to download paid or free software, add-on content, or passes.

How to Obtain Download Codes

  • You can purchase Download Cards containing a download code at Nintendo retailers or select convenience stores.
  • You can purchase download codes directly from select online stores.

※Purchase methods vary by retailer. Please check the official Nintendo website for more details.
